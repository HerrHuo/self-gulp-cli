# self-gulp-cli

个人gulp架构，适合多页整项目。

### 结构介绍

``` bash
- build(配置文件)

- src(项目文件)

    - activity(例：活动项目)
    
    - include(公共html模板文件)
    
    - js(js文件)
    
    - less(less文件)
    
    - ...
    
- ...
```

### 操作指南

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:86
npm run dev

# build for production
npm run build
```
