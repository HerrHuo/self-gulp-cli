# self-gulp-cli

gulp架构

### 结构介绍

``` bash
- build(配置文件)

- src(项目文件)
    
    - modules(公共html模块文件)
    
    - js(js文件)

        - modules(公共js模块文件)
    
    - css(css文件)

        - modules(公共css模块文件)
    
    - ...
    
- ...
```

### 操作指南

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:86
npm run dev

# build for production
npm run build
```
